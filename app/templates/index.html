Welcome to the Measurement Tool for Vancouver Storefronts

This tool will take in a submitted file and return a calculated storefront sizing.

his is done by checking to make sure that the storefront's centroid is located in the polygon of the floor sizing.

If multiple storefronts (centroids) exist in a building or polygon, then the area will be divided by the number of centroids, giving it the average.
